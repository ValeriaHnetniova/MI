# 🎒 Knapsack Problem Solver (0/1)

[![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)](https://www.python.org/downloads/)
[![UI](https://img.shields.io/badge/UI-CustomTkinter-orange.svg)](https://github.com/TomSchimansky/CustomTkinter)
[![Pattern](https://img.shields.io/badge/Pattern-Strategy-green.svg)](https://refactoring.guru/design-patterns/strategy)

**Knapsack Problem Solver** — це інтелектуальна система для моделювання та розв'язання класичної задачі комбінаторної оптимізації. Програма дозволяє порівнювати п'ять різних алгоритмічних підходів, візуалізувати процес прийняття рішень та аналізувати складність обчислень.

---

## ✨ Основні особливості

* **Мультиалгоритмічне ядро:** Підтримка 5 методів розв'язання (від повного перебору до методу гілок і меж).
* **Інтерактивна візуалізація:** Автоматична побудова матриці станів для алгоритму динамічного програмування (DP).
* **Сучасний UX/UI:** Темна тема, адаптивний дизайн та інтуїтивно зрозуміле керування на базі `CustomTkinter`.
* **Надійна валідація:** Розумна обробка некоректного вводу даних без зупинки програми.
* **Повний звіт:** Детальна розшифровка результатів (максимальна цінність, підсумкова вага, перелік обраних індексів).

---

## 🧠 Реалізовані алгоритми

| # | Метод | Тип | Особливість |
|:---|:---|:---|:---|
| 1 | **Brute Force** | Точний | Перевірка всіх $2^n$ комбінацій. |
| 2 | **Recursive** | Точний | Класичний обхід дерева рішень «зверху-вниз». |
| 3 | **Greedy** | Наближений | Швидке сортування за питомою цінністю ($v/w$). |
| 4 | **Dynamic Programming** | Точний | Оптимальний метод з використанням пам'яті та візуалізацією матриці. |
| 5 | **Branch and Bound** | Точний | Розумний обхід дерева з відсіканням неперспективних гілок. |

---

## 🏗️ Архітектура системи

Проєкт реалізовано за принципами **чистої архітектури** та розділення відповідальності:

* **Model (`solvers.py`):** Ізольована математична логіка алгоритмів.
* **View (`gui_view.py`):** Графічний інтерфейс та логіка візуалізації таблиць.
* **Controller (`main_controller.py`):** Сполучна ланка, що реалізує патерн **Strategy** для динамічного вибору алгоритму.
* **Data Provider (`data_provider.py`):** Сервіс валідації, очищення даних та парсингу вводу.

## 📊 Демонстрація роботи

### Тестовий сценарій (Варіант №11):
- **Місткість (W):** `24`
- **Ваги:** `9, 3, 3, 3, 5, 8, 9`
- **Цінності:** `11, 10, 6, 4, 15, 3, 12`

**Результат:**
Програма підтверджує абсолютний оптимум: **Цінність = 47, Вага = 23**.
